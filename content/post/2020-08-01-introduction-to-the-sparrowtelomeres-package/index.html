Overview

This package contains 3 things

  1. A replication of stastical results in Meillere et al (2015) Traffic noise exposure affects telomere length in nestling house sparrows. Bilogy Letters 11: 20150559. https://doi.org/10.1098/rsbl.2015.0559
  2. Readings and exercises based on Meillere et al. (2015) designed for introductory biology courses. Some of these use a reduced dataset to simplify analyses.
  3. Examples of data visualization techinques using the data from Meillere et al. (2015)

Abridged & Annotated Version of Meillere et al (2015)

The following types of chagnes were made to the abridged and annotated version of the paper:

  1. Overall the paper was shortened, techincal terms removed or annotated with definitions, and techncial language simplified.
  2. The abstract was converted to a structured formated with numbered bullets, which studies have shown increases comprehension.
  3. Added keywords
  4. Simplified the presentation of the methods
  5. Created a subset of the data which yielded similar results but which could be analyzed using basic statistics instead of mixed models.
  6. Generated new statistical results and graphs using the data.
